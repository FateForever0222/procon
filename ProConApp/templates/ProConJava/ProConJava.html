ProCon Java

Overview

We use java language to encapsulate the algorithm and realize the software with complete function and friendly interface. In order to facilitate researchers to calculate, check and store the triple conservative results of amino acid sites.

Due to the cross platform nature of Java language, the software can be used in different operating systems. The software divides the basic functions and calculation results into several tabs for users to check. And the important parameters are configured by the user in the menu.
